BISH-BOSH 2.0

How to do:

  1. Choose a positive number to end BishBosh
  2. Choose a positive number to be the divisable number that prints out Bish
  3. Choose another positive number to be the divisable number that prints out Bosh

Rules for BishBosh:

Bish-Bosh lists all numbers between 1 and 100, with some exceptions: If the number is divisible by 3, 'Bish' is listed instead of the number. If the number is divisible by 4, 'Bosh' is listed instead of the number. If the number is divisible by both 3 and 4, "Bish-Bosh" is listed instead of the number. Output Example: 1, 2, Bish, Bosh, 5, Bish, 7, Bosh, Bish, 10, 11, Bish-Bosh, 13, 14, Bish, [...]
